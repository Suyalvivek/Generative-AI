# Prompt Message Structure

Example showing how chat messages are structured with `system` and `user` roles.

## What It Does

`app.js` creates a polite personal assistant through a system message, asks `Who are you?`, and prints the model response.

## Setup

```bash
npm install
```

Required environment variable:

```bash
GROQ_API_KEY=your_groq_api_key
```

## Run

```bash
GROQ_API_KEY=your_groq_api_key node app.js
```
