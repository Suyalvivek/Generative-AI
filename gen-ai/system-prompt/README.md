# System Prompt

Example of using a system prompt to shape the assistant's behavior.

## What It Does

`app.js` asks the model to act as a sentiment analyzer and return a single-word sentiment for a product review.

## Setup

```bash
npm install
```

Required environment variable:

```bash
GROQ_API_KEY=your_groq_api_key
```

## Run

```bash
GROQ_API_KEY=your_groq_api_key node app.js
```
