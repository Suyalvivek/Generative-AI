# LLM Invoke

Minimal example of calling a Groq chat completion from Node.js.

## What It Does

`app.js` sends a single user message, `Hi`, to `llama-3.3-70b-versatile` and prints the assistant response.

## Setup

```bash
npm install
```

Required environment variable:

```bash
GROQ_API_KEY=your_groq_api_key
```

## Run

```bash
GROQ_API_KEY=your_groq_api_key node app.js
```
